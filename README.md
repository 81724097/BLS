#### 更新日志
- 2018年2月17日14:41:55，提交第一个版本，完成了多账户登陆控制，弹幕服务器连接控制，基本流程控制
- 技术栈: Nodejs / Electron / Vue / element.ui


#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
